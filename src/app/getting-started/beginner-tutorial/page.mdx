# Beginner tutorial

This is the beginner tutorial course for Teo. By reading the chapters in this 
tutorial, developer learns the concepts, usages and features of Teo. It's 
recommended for developers who use Teo for the first time. For advanced 
Teo developers, read our [Guides](/guides) for an in-depth understanding 
of Teo and best practices. If you are seeking for API reference and CLI reference,
head to [Reference](/reference).
