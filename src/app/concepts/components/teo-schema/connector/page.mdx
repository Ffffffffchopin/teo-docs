# Connector

The connector specifies which database to connect. A Teo app can have only
one connector. Teo currently supports MySQL, PostgreSQL, SQLite and MongoDB.

## Example

To connect to a database, specify a
[provider](/reference/api-reference/teo-schema-reference#types) <span class="concept">REFERENCE</span>
value and a connection URL.

```teo
connector {
  provider: .mysql,
  url: "mysql://127.0.0.1:3306/helloteo"
}
```
