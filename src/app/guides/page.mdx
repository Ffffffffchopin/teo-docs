# Guides

This book of the documentation is a collection of topics that guides developers
to elevate Teo to get things done. If you want to know what makes up Teo and
what the components are, read [Concepts](/docs/concepts). For API documentation, 
read [API Reference](/docs/reference/api-reference).

The guides are organized into the following topics:

* [Server guides](/docs/guides/server-guides)
  * [Data modeling](/docs/guides/server-guides/data-modeling)
  * [Route handlers](/docs/guides/server-guides/route-handlers)
  * [Middlewares](/docs/guides/server-guides/middlewares)
  * [Namespaces](/docs/guides/server-guides/namespaces)
* [Query client guides](/docs/guides/query-client-guides)
  - [CRUD](/docs/guides/query-client-guides/crud)
  - [Filtering and sorting](/docs/guides/query-client-guides/filtering-and-sorting)
  - [Pagination](/docs/guides/query-client-guides/pagination)
  - [Selecting fields](/docs/guides/query-client-guides/selecting-fields)
  - [Relation queries](/docs/guides/query-client-guides/relation-queries)
  - [Aggregation, grouping and summarizing](/docs/guides/query-client-guides/aggregation-grouping-summarizing)
