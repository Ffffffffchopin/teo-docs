# Guides

This book of the documentation is a collection of topics that guides developers
to elevate Teo to get things done. If you want to know what makes up Teo and
what the components are, read [Concepts](/concepts). For API documentation, 
read [API Reference](/reference/api-reference).

The guides are organized into the following topics:

* [Server guides](/guides/server-guides)
  * [Data modeling](/guides/server-guides/data-modeling)
  * [Route handlers](/guides/server-guides/route-handlers)
  * [Middlewares](/guides/server-guides/middlewares)
  * [Namespaces](/guides/server-guides/namespaces)
* [Query client guides](/guides/query-client-guides)
  - [CRUD](/guides/query-client-guides/crud)
  - [Filtering and sorting](/guides/query-client-guides/filtering-and-sorting)
  - [Pagination](/guides/query-client-guides/pagination)
  - [Selecting fields](/guides/query-client-guides/selecting-fields)
  - [Relation queries](/guides/query-client-guides/relation-queries)
  - [Aggregation, grouping and summarizing](/guides/query-client-guides/aggregation-grouping-summarizing)
